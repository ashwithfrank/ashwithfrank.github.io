Tower of Hanoi

An interactive solver that breaks down the Tower of Hanoi puzzle step-by-step — built with vanilla JavaScript using recursion.

Live Solo Project Web App JavaScript

Overview

Tower of Hanoi is a classic mathematical puzzle where you move a stack of disks from one peg to another, following one rule: never place a larger disk on a smaller one. This web app takes any number of disks as input and instantly outputs every move required to solve it, along with the total move count.

How It Works

      

        

          
🔄

          
Recursive Algorithm

          
Uses a classic recursive function that breaks the problem down to a single-disk base case.

        

        

          
⚡

          
Instant Results

          
Updates live as you type — no button press needed. Results appear in real time.

        

        

          
🔢

          
Move Counter

          
Displays total moves using the formula 2ⁿ−1, proving the mathematical minimum.

        

        

          
📋

          
Step-by-Step Output

          
Every individual disk move is printed in order, showing source and destination peg.

What I Learned

      

        This project helped me get comfortable with recursion as a real problem-solving tool. Reducing a complex multi-step puzzle to a clean three-line recursive function was a great lesson in how elegant code can be when the logic is right. I also focused on clean UI — the glassmorphism styling and live input make it feel polished despite being pure vanilla JS.
